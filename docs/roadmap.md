# Roadmap

Planned features and improvements for hive.

## :material-monitor-dashboard: Web UI

- Session replay with message-by-message playback
- File lineage visualization (graph view of sessions, commits, and files)
- Project dashboard with quality trends and token usage charts
- Search interface with filters and highlighted results

## :material-shield-lock: Authentication and Access Control

- API key authentication for the team server
- Role-based access (admin, member, read-only)
- Per-project access control lists
- SSO / OAuth integration

## :material-brain: AI-Powered Session Analysis

- Automatic session categorization (bug fix, feature, refactor, exploration)
- Quality coaching: detect patterns like excessive corrections or long tool loops
- Session summaries generated by LLM instead of first-message heuristic
- Team productivity insights and trends

## :material-puzzle: Additional Capture Adapters

- **Cursor** -- ingest Cursor AI sessions via workspace logs
- **Codex** -- capture OpenAI Codex CLI sessions
- **ChatGPT** -- import exported ChatGPT conversations
- **GitHub Copilot** -- capture Copilot Chat interactions from VS Code

## :material-webhook: Webhooks

- Configurable webhook endpoints triggered on session capture
- Slack / Discord notifications for team activity
- CI/CD integration (post session links on PRs)

## :material-database: PostgreSQL Backend

- Optional PostgreSQL storage for larger teams
- Connection pooling and concurrent write support
- Migration tooling from SQLite to PostgreSQL
