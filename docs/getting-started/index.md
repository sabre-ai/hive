# Getting Started

Pick your setup level — each stage builds on the last. Stop at any point and you have a working setup.

| Stage | What you get |
|-------|-------------|
| [Solo Mode](solo-mode.md) | Local capture, search, and MCP for Claude Code |
| [Claude Desktop](desktop.md) | Unified history across Code and Desktop |
| [Team Server](team-server.md) | Shared searchable history for your team |

Having trouble? See [Troubleshooting](troubleshooting.md) or the [FAQ](faq.md).
