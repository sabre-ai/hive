# Getting Started

Install hive with a single command:

```bash
curl https://sabre-ai.github.io/hive/install.sh | bash
```

This installs the hive CLI and configures the MCP server for both Claude Code and Claude Desktop. Then enable capture for each project with `hive init`.

|-------|-------------|
| [Quick Start](solo-mode.md) | Install, local capture, search, and MCP for Claude Code + Desktop |
